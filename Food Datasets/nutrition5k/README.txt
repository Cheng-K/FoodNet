Readme
=======

This folder contains the preprocessing script for Nutrition5k dataset. 
Nutrition5k dataset can be downloaded at : https://github.com/google-research-datasets/Nutrition5k

Extract and put the content from the downloaded dataset into this folder.
The structure of this folder should look like below :
    |
    |- dish_ids
    |- imagery
    |- metadata
    |- preprocess_nutrition5k.ipynb
    |- README.txt
