Readme
=======

This folder contains the preprocessing script for Recipes5k dataset. 
Recipes5k dataset can be downloaded at : http://www.ub.edu/cvub/recipes5k/

Extract and put the content from the downloaded dataset into this folder.
The structure of this folder should look like below :
    |
    |- annotations
    |- images
    |- ingredients_simplification
    |- preprocess_recipes5k.ipynb
    |- README.txt
