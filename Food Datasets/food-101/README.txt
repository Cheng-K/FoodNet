Readme
=======

This folder contains the preprocessing script for Food101 dataset. 
Food101 dataset can be downloaded at : https://data.vision.ee.ethz.ch/cvl/datasets_extra/food-101/

Extract and put the images and meta folder from the downloaded dataset into this folder.
The structure of this folder should look like below :
    |
    |- images
    |- meta
    |- preprocess_food101.ipynb
    |- README.txt
